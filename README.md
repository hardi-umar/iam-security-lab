# Identity and Access Management (IAM) Security Lab

## Project Overview

This project demonstrates practical Identity and Access Management (IAM) concepts using Microsoft Entra ID (Azure AD) and enterprise security principles.

The objective of this lab is to simulate how organizations manage user identities, authentication, authorization, and access governance within cloud environments.

## Technologies

- Microsoft Entra ID (Azure AD)
- Microsoft Authenticator
- Role-Based Access Control (RBAC)
- Conditional Access
- Multi-Factor Authentication (MFA)
- Group-Based Access Control
- Single Sign-On (SSO)
- Microsoft Intune
- Microsoft Defender

## Lab Objectives

- Create and manage users
- Create security groups
- Assign role-based permissions
- Configure MFA
- Implement Conditional Access policies
- Demonstrate user provisioning and deprovisioning
- Document IAM security controls
- Capture screenshots as evidence

## Skills Demonstrated

- Identity Governance
- Access Control
- Authentication
- Authorization
- Security Operations
- Cloud Security
- Microsoft Security Stack
