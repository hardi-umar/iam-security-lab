# Lab 01 - User Provisioning and Identity Management

## Objective

Demonstrate the process of creating and managing user identities within Microsoft Entra ID.

## Activities

- Create a new user account
- Assign user attributes
- Assign group membership
- Review assigned permissions
- Verify account creation

## Security Concepts

- Identity Lifecycle Management
- User Provisioning
- Access Governance
- Least Privilege
